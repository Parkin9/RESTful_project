# RESTful_project

The simple RESTful microservice. It gets a Json from NewsApi.org api, formats a little and sends it as a Json too.
The project contains a little Front-end client. It displays provided Json.

BUILD'n'RUN project<br/>
- For Unix:<br/>
1) Open a project's main folder in a terminal.<br/>
2) Use a command: "./mvnw spring-boot:run".
<br/><br/>

- For Windows:<br/>
1) Open a project's main folder in a console.<br/>
2) Use a command: ".\mvnw.cmd spring-boot:run".
